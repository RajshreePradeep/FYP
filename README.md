# FYP
Brain tumour classification system using CNN (VGG16) to detect Glioma, Meningioma, Pituitary and No Tumour from MRI images. Uses Grad-CAM for explainable AI visualisation and an LLM for automated report generation. End-to-end pipeline combining prediction, explanation, and reporting.
This project presents a brain tumour classification system using a Convolutional Neural Network (VGG16 with transfer learning) to detect Glioma, Meningioma, Pituitary tumour, and No tumour from MRI images. The model is trained on the Brain Tumor MRI dataset (Kabir, 2025) available at https://doi.org/10.17632/zwr4ntf94j.1. Grad-CAM is used for visual explainability by highlighting important regions in MRI scans. A large language model is also integrated to generate simple radiology-style reports from predictions. The system combines image classification, explainable AI, and automated reporting in a single pipeline.
